How it works

The premium route is configured with canActivateFeatureFlag({{ '{' }} flagKey: 'premium-feature-x', context: () => inject(FeatureFlagsDemoSessionService).context(), redirectTo: '/upgrade' {{ '}' }}). The guard evaluates premium-feature-x against the current persona's context at navigation time, redirecting to upgrade when denied.
