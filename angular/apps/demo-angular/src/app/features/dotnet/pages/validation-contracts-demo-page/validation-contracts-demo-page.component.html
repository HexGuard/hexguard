Public types

The library exports these core types:

  • ValidationError — Field, Code, Message, IsFieldError
  • ValidationResult — Errors, TraceId, IsValid, field filtering
  • ValidationResultBuilder — fluent error construction
  • FieldPath — Child, Index, IndexChild, GetParent, GetLeaf
  • ValidationErrorCode — Required, InvalidFormat, OutOfRange, etc.
  • ValidationResultProblemDetails — RFC 9457 adapter
