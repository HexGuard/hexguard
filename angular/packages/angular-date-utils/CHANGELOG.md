# Changelog

## 0.1.0

- Initial release.
- Pure functions for relativeTime, shortRelativeTime, exactRelativeTime, compactDate, compactDateTime.
- Business-day helpers: isWeekend, addBusinessDays, businessDaysBetween.
- Duration helpers: durationBetween, formatDuration, ageInYears.
- DateRange model class with validation, containment, overlap, and preset factories.
- injectDateUtils() DI wrapper with locale detection.
