# Changelog
## 0.1.0 — 2026-06-22
Initial release of `@hexguard/angular-form-drafts`.
### Features
- `injectFormDraft()` — localStorage-backed form draft persistence with signals
- Auto-save with configurable debounce
- Restore, clear, and check for existing drafts
- Draft expiry for stale data cleanup
- Draft metadata (saved timestamp, expiry time)
