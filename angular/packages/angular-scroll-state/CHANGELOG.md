# Changelog

## 0.1.0 — 2026-06-23

Initial release of `@hexguard/angular-scroll-state`.

### Features

- `injectScrollState()` — scroll position tracking with save/restore
- `inInfiniteScroll()` — IntersectionObserver-based infinite-scroll detection
- `inScrollSpy()` — section tracking via IntersectionObserver
- `scrollTo()` — imperative smooth-scroll helper
- `requestAnimationFrame`-throttled scroll listeners
- Configurable scroll container and debounce
- Automatic cleanup via DestroyRef
