# Changelog
## 0.1.0 — 2026-06-22
Initial release of `@hexguard/angular-file-picker`.
### Features
- `injectFilePicker()` — headless file selection with signal-based state
- File type validation via `accept` filter
- File size validation via `maxFileSize` option
- Multiple/single file modes
- Read file contents as text, data URL, or ArrayBuffer
- Drag-and-drop zone support with `acceptDrop()` helper
