# AI Workflow

Canonical AI workflow docs live under `docs/.ai/`.

Use these files instead of maintaining a second workflow surface under `docs/ai/`:

- `docs/.ai/README.md`: repo-wide AI workflow guide
- `docs/.ai/backlog.md`: short roadmap queue
- `docs/.ai/backlog/README.md`: detailed backlog structure and status guidance
- `docs/.ai/decisions/README.md`: accepted planning and workflow decisions
- `docs/.ai/release-checklist.md`: release-specific execution checklist

This folder remains only as a compatibility path for older links.
