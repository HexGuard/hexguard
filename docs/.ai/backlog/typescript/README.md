# TypeScript Space

This space holds framework-agnostic TypeScript/JavaScript packages. These packages work in any JS runtime (Node.js, Deno, Bun, browsers) and have zero framework dependencies — no Angular, no React, no Vue.

## Constraints

- Zero runtime dependencies (built on the platform).
- ESM-first, tree-shakeable.
- TypeScript-first with exported types.
- Works in Node.js, Deno, Bun, and browsers.
- Package ID format: `@hexguard/ts-{name}` or `@hexguard/{name}`.

## Target Audience

- Full-stack TypeScript developers
- Node.js backend developers
- Library authors
- Any TypeScript project, framework-agnostic
