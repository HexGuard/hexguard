# AI Prompting Cross-Stack Package Pair

## Summary

AI prompt management across the stack — server-side template engine + client-side prompt editor with variable interpolation, test-run, and streaming output.

### .NET (`HexGuard.Prompting`)
Prompt template storage, rendering, token estimation, OpenAI/Anthropic adapters.

### Angular (`@hexguard/angular-prompt`)
Template CRUD, variable editor, live preview, test-run with streaming via `angular-chat`.
