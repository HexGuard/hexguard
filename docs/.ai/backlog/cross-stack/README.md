# Cross-Stack Space

This space holds shared package families and contract work that intentionally spans Angular and .NET.

Each brief should define the shared semantics first, then record the Angular deliverable, the .NET deliverable, release-coupling expectations, and the sample or demo flow that proves the pair works end to end.

For now, treat these briefs as parked sidenotes unless the active Angular and .NET spaces both need the same contract at the same time.
